> Sub-plan of `.claude/PRPs/plans/kmp-claude-code-remote.plan.md`, Phase W1, Task 1.5.
> Prerequisites: W1.1 (✅ merged — WireMessage/CCEvent), W1.3 (✅ merged — ClaudeProcess),

**Goal:** Ship the end-to-end security handshake so a paired mobile client can open a single WSS session to the local relay, with Ed25519-signed `ClientHello` + 6-digit one-time pairing codes + nonce-replay defense + revoke CLI. Plumbs the output of W1.4's `ApprovalBridge` into an actual WebSocket so the relay becomes a real mobile-facing server.

**Architecture:** Ports & Adapters per `openspec/project.md`. Pure-Kotlin crypto primitive lives in `shared/core/crypto/` (Ed25519 key + sign/verify — `expect`/`actual` for platform-native entropy). `shared/core/pairing/` hosts session identity types (`DeviceId`, `PairingCode`, `ClientHello`, `SessionGreeting`). `shared/adapters/transport/` gets a Ktor WebSocket `RelayClient` that signs hellos and exchanges `WireMessage` frames. `relay/pairing/` adds `PairingService` (code issue/redeem + pubkey registry + revoke) and `ClientHelloVerifier` (sig + clock-skew + nonce replay). `relay/server/RelayServer.kt` wires everything into one Ktor Netty WS endpoint and bridges the existing `ApprovalBridge.outbound: SharedFlow<WireMessage>` onto the socket. A new `relay-cli pair` / `relay-cli revoke` subcommand surfaces the admin ops.

**Tech Stack:** Kotlin 2.1 KMP · kotlinx.coroutines · kotlinx.serialization · kotlinx.datetime · **Ktor 3.0 WebSockets** (client for shared, server for relay) · pure-Kotlin Ed25519 (no JNI — we inline a minimal verified impl in `shared/core/crypto/` from `ed25519-elisabeth` equivalents; see MIRROR-2) · kotlin.test + kotlinx-coroutines-test. **No new dependencies** beyond libs already in `libs.versions.toml` (Ktor client/server/websockets already aliased in W0).

## Patterns to Mirror

### MIRROR-1: Ports & Adapters (project.md §Architecture Patterns)

`shared/core/**` stays framework-free — no Ktor imports. The `TransportPort` sealed interface lives in `shared/ports/`, with the Ktor-backed `RelayClient` in `shared/adapters/transport/`. This matches how W1.4 separated `ApprovalStore` (port in `relay/approval/`) from `InMemoryApprovalStore` (adapter).

### MIRROR-2: Pure-Kotlin Ed25519 (no libsodium, no JNI)

Rationale: iOS `platform.CryptoKit` exposes Ed25519 only from iOS 17+; Android has `java.security.Signature("Ed25519")` from API 34+ — we need API 26. Rather than stacking two platform backends now, we port the reference RFC 8032 implementation into `shared/core/crypto/Ed25519.kt` (pure Kotlin, constant-time field arithmetic on `LongArray`). **Source to mirror**: `str4d/ed25519-elisabeth` (Java) — we translate only the subset we need (sign / verify + seed→keypair), ≈ 400 LOC total. Unit tested against **RFC 8032 §7.1 test vectors** to guarantee correctness before it's trusted for protocol work.

Platform `expect`/`actual` is narrowed to one thing only: `SecureRandom` (16 bytes of entropy for nonce + 32 bytes for keypair seed). `jvmMain`/`androidMain` use `java.security.SecureRandom`; `iosMain` uses `platform.Security.SecRandomCopyBytes`.

### MIRROR-3: ClientHello framing (spec: `pairing/spec.md` §Ed25519 会话签名)

```jsonc
// First WS frame after upgrade, Text payload:
{
  "op": "hello",
  "deviceId": "dev_Zxy...",              // base64url of pubkey-fp (16B sha256 prefix)
  "sessionId": "S1",
  "timestampMs": 1745000000000,
  "nonce": "b64url(16 random bytes)",
  "sig": "b64url(Ed25519((sessionId||'|'||timestampMs||'|'||nonce).utf8))"
}
```

Response from relay:

```jsonc
{ "op": "hello.ok", "serverTimeMs": 1745000000123, "protocolVersion": 1 }
// or on failure:
{ "op": "hello.err", "reason": "skew|nonce|sig|revoked|unpaired" }
```

Both shapes join the existing `WireMessage` hierarchy — reuses the `WireMessageSerializer` + `ProtocolJson` already shipped in W1.1. The discriminator is still `"op"`.

### MIRROR-4: Virtual-clock + fake-entropy seams (pattern from W1.4 `TimeSeam`)

Tests drive time and randomness deterministically:

```kotlin
// relay/src/main/kotlin/io/ohmymobilecc/relay/pairing/ClockSeam.kt
public interface ClockSeam { public fun nowMs(): Long }
public object SystemClockSeam : ClockSeam { override fun nowMs(): Long = kotlinx.datetime.Clock.System.now().toEpochMilliseconds() }

// shared/src/commonMain/kotlin/io/ohmymobilecc/core/crypto/RandomSource.kt
public interface RandomSource { public fun nextBytes(size: Int): ByteArray }
public expect fun platformSecureRandom(): RandomSource
```

Tests pass `FakeClock(start = 1_000_000_000L)` and `FakeRandom(bytesOf(0x00, 0x01, ...))`. Same pattern worked cleanly in W1.4.

### MIRROR-5: Nonce cache — bounded, monotonic-time evict

Spec mandates 10-minute window. Implementation: `ConcurrentHashMap<String, Long>` keyed by b64url nonce, value = insertion nowMs. On each accept, purge entries where `nowMs - insertedAt > 600_000`. Bounded at 10_000 entries (≈ 600 MB worst case is impossible — nonces are 16 bytes); if a malicious peer floods faster than eviction keeps up, the map is dropped and a warn logged. Same pattern as Redis SETEX but in-process.

### MIRROR-6: Single-connection enforcement (protocol spec §传输语义)

`RelayServer` holds `ConcurrentHashMap<SessionId, WebSocketSession>`. Second connect for same `sessionId` → close with code 1013 (`try again later`). This matches the spec's explicit scenario and prevents split-brain with `ApprovalBridge`'s single outbound `SharedFlow` subscriber.

### MIRROR-7: Test structure

Consistent with W1.3/W1.4: kotlin.test + JUnit Platform. `runTest` for time-sensitive code (timeout, skew, replay window eviction). Real Ktor `testApplication` for the server end-to-end — no raw socket juggling; Ktor's test harness exposes a virtual `HttpClient` that speaks WS without binding a port.

                "085ac1e43e15996e458f3613d0f11d8c387b2eaeb4302aeeb00d291612bb0c00",

### Task 2 — [GREEN] Implement pure-Kotlin Ed25519

**Files:**
- Create: `shared/src/commonMain/kotlin/io/ohmymobilecc/core/crypto/Ed25519.kt`

**Scope note:** ≈ 400 LOC. Pattern this after str4d's `ed25519-elisabeth` (Java reference, MIT license, header credit in KDoc). The structure is:
1. `FieldElement` on 2²⁵⁵ − 19 with radix-2⁶⁴ `LongArray(5)` limbs.
2. `Scalar` mod ℓ (group order).
3. `EdwardsPoint` with extended projective `(X, Y, Z, T)` coords + `doubleAndAdd` table.
4. `Ed25519.keypair(seed)` = SHA-512(seed)[0..31] clamped → scalar → `scalar * B` = pk.
5. `Ed25519.sign(sk, m)` = RFC 8032 §5.1.6.
6. `Ed25519.verify(pk, m, sig)` = §5.1.7 — rejects non-canonical `R`, `S ≥ ℓ`.

SHA-512 comes from a tiny pure-Kotlin `Sha512` helper in the same file (≈ 80 LOC, test-vector-checked inside `Ed25519Test`'s fixture).

- [ ] **Step 1: Implement the code**

Create the file. (Full implementation is too long to inline here; agent should port from `str4d/ed25519-elisabeth` — permissive license — or from RFC 8032 §6 reference code. The file must:
- expose `object Ed25519 { fun keypair(seed: ByteArray): Ed25519KeyPair; fun sign(...); fun verify(...) }`
- define `data class Ed25519KeyPair(val secretKey: ByteArray, val publicKey: ByteArray)` — secretKey is 64 bytes (seed || pk) per RFC §5.1.5.
- be pure-Kotlin, no `java.*` imports, no `!!`, no wildcard imports.
- constant-time comparison for signature check (`subtle.ConstantTime.equal`-style loop).)

// shared/src/commonMain/kotlin/io/ohmymobilecc/core/crypto/Base64Url.kt
package io.ohmymobilecc.core.crypto

public object Base64Url {
    private const val ALPHABET = "ABCDEFGHIJKLMNOPQRSTUVWXYZabcdefghijklmnopqrstuvwxyz0123456789-_"

    public fun encode(bytes: ByteArray): String {
        if (bytes.isEmpty()) return ""
        val sb = StringBuilder((bytes.size * 4 + 2) / 3)
        var i = 0
        while (i + 3 <= bytes.size) {
            val n = (bytes[i].toInt() and 0xFF shl 16) or
                (bytes[i + 1].toInt() and 0xFF shl 8) or
                (bytes[i + 2].toInt() and 0xFF)
            sb.append(ALPHABET[(n ushr 18) and 0x3F])
            sb.append(ALPHABET[(n ushr 12) and 0x3F])
            sb.append(ALPHABET[(n ushr 6) and 0x3F])
            sb.append(ALPHABET[n and 0x3F])
            i += 3
        }
        val rem = bytes.size - i
        if (rem == 1) {
            val n = bytes[i].toInt() and 0xFF shl 16
            sb.append(ALPHABET[(n ushr 18) and 0x3F])
            sb.append(ALPHABET[(n ushr 12) and 0x3F])
        } else if (rem == 2) {
            val n = (bytes[i].toInt() and 0xFF shl 16) or
                (bytes[i + 1].toInt() and 0xFF shl 8)
            sb.append(ALPHABET[(n ushr 18) and 0x3F])
            sb.append(ALPHABET[(n ushr 12) and 0x3F])
            sb.append(ALPHABET[(n ushr 6) and 0x3F])
        }
        return sb.toString()
    }

    public fun decode(s: String): ByteArray {
        if (s.isEmpty()) return ByteArray(0)
        val out = ByteArray((s.length * 3) / 4)
        var bits = 0
        var acc = 0
        var o = 0
        for (c in s) {
            val v = ALPHABET.indexOf(c)
            require(v >= 0) { "invalid base64url char: $c" }
            acc = (acc shl 6) or v
            bits += 6
            if (bits >= 8) {
                bits -= 8
                out[o++] = ((acc ushr bits) and 0xFF).toByte()
            }
        }
        return if (o == out.size) out else out.copyOf(o)
    }
}

package io.ohmymobilecc.relay.pairing

import io.ohmymobilecc.core.crypto.RandomSource
import io.ohmymobilecc.core.pairing.DeviceId
import java.util.concurrent.ConcurrentHashMap

public class PairingService(
    private val clock: ClockSeam,
    private val random: RandomSource,
    private val registry: PubkeyRegistry,
) {
    private val issued = ConcurrentHashMap<String, PairingCode>()

    public fun issueCode(): PairingCode {
        val bytes = random.nextBytes(4)
        val n = ((bytes[0].toInt() and 0xFF) shl 24) or
            ((bytes[1].toInt() and 0xFF) shl 16) or
            ((bytes[2].toInt() and 0xFF) shl 8) or
            (bytes[3].toInt() and 0xFF)
        val digits = ((n.toLong() and 0xFFFFFFFFL) % 1_000_000L).toString().padStart(6, '0')
        val code = PairingCode(digits = digits, issuedAtMs = clock.nowMs())
        issued[digits] = code
        return code
    }

    public fun redeem(digits: String, publicKey: ByteArray): DeviceId {
        val code = issued.remove(digits) ?: error("unknown pairing code")
        check(code.isValid(clock.nowMs())) { "pairing code expired" }
        val deviceId = DeviceId.fromPublicKey(publicKey)
        registry.register(deviceId, publicKey, clock.nowMs())
        return deviceId
    }

    public fun revoke(deviceId: DeviceId) {
        registry.revoke(deviceId, clock.nowMs())
    }
}

package io.ohmymobilecc.relay.pairing

import java.util.LinkedHashMap

/**
 * Bounded LRU with TTL eviction. Thread-safety via synchronized blocks —
 * this is only touched from the single server accept coroutine on each
 * ClientHello, so contention is minimal.
 */
public class NonceCache(
    private val maxEntries: Int = 10_000,
    private val ttlMs: Long = 10 * 60 * 1000L,
) {
    private val map = object : LinkedHashMap<String, Long>(16, 0.75f, true) {
        override fun removeEldestEntry(eldest: Map.Entry<String, Long>) = size > maxEntries
    }

    @Synchronized
    public fun offer(nonce: String, nowMs: Long): Boolean {
        purgeExpired(nowMs)
        val existing = map[nonce]
        if (existing != null && (nowMs - existing) < ttlMs) return false
        map[nonce] = nowMs
        return true
    }

    private fun purgeExpired(nowMs: Long) {
        val it = map.entries.iterator()
        while (it.hasNext()) {
            val e = it.next()
            if ((nowMs - e.value) >= ttlMs) it.remove() else break
        }
    }
}

package io.ohmymobilecc.relay.pairing

import io.ohmymobilecc.core.crypto.Base64Url
import io.ohmymobilecc.core.crypto.Ed25519
import io.ohmymobilecc.core.pairing.DeviceId
import io.ohmymobilecc.core.pairing.HelloCodec
import io.ohmymobilecc.core.protocol.WireMessage
import kotlin.math.abs

public sealed interface VerifyResult {
    public data class Ok(val deviceId: DeviceId) : VerifyResult
    public data class Err(val reason: String) : VerifyResult
}

public class ClientHelloVerifier(
    private val registry: PubkeyRegistry,
    private val nonceCache: NonceCache,
    private val clock: ClockSeam,
    private val skewToleranceMs: Long = 60_000L,
) {
    public fun verify(hello: WireMessage.ClientHello): VerifyResult {
        val now = clock.nowMs()
        if (abs(now - hello.timestampMs) > skewToleranceMs) return VerifyResult.Err("skew")

        val device = registry.find(DeviceId(hello.deviceId)) ?: return VerifyResult.Err("unpaired")
        if (device.revokedAtMs != null) return VerifyResult.Err("revoked")

        if (!nonceCache.offer(hello.nonce, now)) return VerifyResult.Err("nonce")

        val canonical = HelloCodec.canonicalSigningInput(
            hello.sessionId, hello.timestampMs, hello.nonce,
        ).encodeToByteArray()

        val sigBytes = runCatching { Base64Url.decode(hello.sig) }.getOrElse { return VerifyResult.Err("sig") }
        if (sigBytes.size != 64) return VerifyResult.Err("sig")

        val valid = Ed25519.verify(device.publicKey, canonical, sigBytes)
        return if (valid) VerifyResult.Ok(device.deviceId) else VerifyResult.Err("sig")
    }
}

package io.ohmymobilecc.relay.server

import java.util.concurrent.ConcurrentHashMap

public class SingleConnectionRegistry {
    private val slots = ConcurrentHashMap<String, Long>()

    public fun claim(sessionId: String, token: Long): Boolean =
        slots.putIfAbsent(sessionId, token) == null

    public fun release(sessionId: String, token: Long) {
        slots.remove(sessionId, token)
    }

    public fun currentToken(sessionId: String): Long? = slots[sessionId]
}

import kotlinx.serialization.decodeFromString
import kotlinx.serialization.encodeToString

public object RelayServer {
    private val tokenCounter = AtomicLong(0)

    public fun install(
        app: Application,
        registry: PubkeyRegistry,
        nonceCache: NonceCache,
        clock: ClockSeam,
        outbound: MutableSharedFlow<WireMessage>,
        onInbound: suspend (WireMessage) -> Unit,
        connectionRegistry: SingleConnectionRegistry = SingleConnectionRegistry(),
    ) {
        val verifier = ClientHelloVerifier(registry, nonceCache, clock)

        app.routing {
            webSocket("/ws") {
                val first = try {
                    incoming.receive()
                } catch (_: ClosedReceiveChannelException) {
                    return@webSocket
                }
                if (first !is Frame.Text) {
                    close(CloseReason(CloseReason.Codes.VIOLATED_POLICY, "text frame required"))
                    return@webSocket
                }

                val msg = runCatching {
                    ProtocolJson.default.decodeFromString<WireMessage>(first.readText())
                }.getOrElse {
                    send(Frame.Text(ProtocolJson.default.encodeToString<WireMessage>(
                        WireMessage.HelloErr("malformed"),
                    )))
                    close(CloseReason(CloseReason.Codes.CANNOT_ACCEPT, "malformed"))
                    return@webSocket
                }

                val hello = msg as? WireMessage.ClientHello ?: run {
                    send(Frame.Text(ProtocolJson.default.encodeToString<WireMessage>(
                        WireMessage.HelloErr("expected-hello"),
                    )))
                    close(CloseReason(CloseReason.Codes.VIOLATED_POLICY, "expected hello"))
                    return@webSocket
                }

                when (val r = verifier.verify(hello)) {
                    is VerifyResult.Err -> {
                        send(Frame.Text(ProtocolJson.default.encodeToString<WireMessage>(
                            WireMessage.HelloErr(r.reason),
                        )))
                        close(CloseReason(CloseReason.Codes.VIOLATED_POLICY, r.reason))
                        return@webSocket
                    }
                    is VerifyResult.Ok -> Unit
                }

                val token = tokenCounter.incrementAndGet()
                if (!connectionRegistry.claim(hello.sessionId, token)) {
                    send(Frame.Text(ProtocolJson.default.encodeToString<WireMessage>(
                        WireMessage.HelloErr("duplicate-session"),
                    )))
                    close(CloseReason(CloseReason.Codes.TRY_AGAIN_LATER, "duplicate session"))
                    return@webSocket
                }

                try {
                    send(Frame.Text(ProtocolJson.default.encodeToString<WireMessage>(
                        WireMessage.HelloOk(serverTimeMs = clock.nowMs(), protocolVersion = 1),
                    )))

                    val outboundJob = launch {
                        outbound.onEach { wm ->
                            send(Frame.Text(ProtocolJson.default.encodeToString<WireMessage>(wm)))
                        }.collect { /* loop */ }
                    }

                    try {
                        for (frame in incoming) {
                            if (frame !is Frame.Text) continue
                            val wm = runCatching {
                                ProtocolJson.default.decodeFromString<WireMessage>(frame.readText())
                            }.getOrNull() ?: continue
                            onInbound(wm)
                        }
                    } finally {
                        outboundJob.cancel()
                    }
                } catch (ce: CancellationException) {
                    throw ce
                } finally {
                    connectionRegistry.release(hello.sessionId, token)
                }
            }
        }
    }
}

package io.ohmymobilecc.adapters.transport

import io.ktor.client.HttpClient
import io.ktor.client.plugins.websocket.DefaultClientWebSocketSession
import io.ktor.client.plugins.websocket.webSocketSession
import io.ktor.websocket.Frame
import io.ktor.websocket.readText
import io.ktor.websocket.send
import io.ohmymobilecc.core.crypto.Base64Url
import io.ohmymobilecc.core.crypto.Ed25519
import io.ohmymobilecc.core.pairing.HelloCodec
import io.ohmymobilecc.core.protocol.ProtocolJson
import io.ohmymobilecc.core.protocol.WireMessage
import io.ohmymobilecc.ports.RelayConfig
import io.ohmymobilecc.ports.TransportPort
import io.ohmymobilecc.ports.TransportSession
import kotlinx.coroutines.flow.Flow
import kotlinx.coroutines.flow.channelFlow
import kotlinx.serialization.decodeFromString
import kotlinx.serialization.encodeToString

public class KtorRelayClient(
    private val httpClient: HttpClient,
) : TransportPort {
    override suspend fun connect(config: RelayConfig): TransportSession {
        val ws: DefaultClientWebSocketSession = httpClient.webSocketSession(config.url)
        val nonce = config.randomNonce()
        val ts = config.clockNowMs()
        val canonical = HelloCodec.canonicalSigningInput(config.sessionId, ts, nonce).encodeToByteArray()
        val sig = Ed25519.sign(config.keyPair.secretKey, canonical)
        val hello = WireMessage.ClientHello(
            deviceId = config.deviceId,
            sessionId = config.sessionId,
            timestampMs = ts,
            nonce = nonce,
            sig = Base64Url.encode(sig),
        )
        ws.send(Frame.Text(ProtocolJson.default.encodeToString<WireMessage>(hello)))

        val firstFrame = ws.incoming.receive()
        if (firstFrame !is Frame.Text) {
            ws.close()
            throw RelayError.Malformed("expected text frame, got $firstFrame")
        }
        val firstMsg = ProtocolJson.default.decodeFromString<WireMessage>(firstFrame.readText())
        when (firstMsg) {
            is WireMessage.HelloOk -> { /* ok */ }
            is WireMessage.HelloErr -> {
                ws.close()
                throw RelayError.HelloRejected(firstMsg.reason)
            }
            else -> {
                ws.close()
                throw RelayError.Malformed(firstFrame.readText())
            }
        }

        return object : TransportSession {
            override val incoming: Flow<WireMessage> = channelFlow {
                for (frame in ws.incoming) {
                    if (frame !is Frame.Text) continue
                    runCatching {
                        ProtocolJson.default.decodeFromString<WireMessage>(frame.readText())
                    }.onSuccess { send(it) }
                }
            }

            override suspend fun send(msg: WireMessage) {
                ws.send(Frame.Text(ProtocolJson.default.encodeToString<WireMessage>(msg)))
            }

            override suspend fun close() { ws.close() }
        }
    }
}

### Edge cases checklist

- [ ] Timestamp exactly at the ±60 000 ms boundary: boundary test in `ClientHelloVerifierTest` (strictly `> skewTolerance` rejects; `== skewTolerance` accepts — document in code comment).
- [ ] Base64Url input of length 1 (invalid — must decode to 0 bytes or fail gracefully).
- [ ] Nonce of empty string: `NonceCache.offer("", now)` should be accepted once but is a client bug — we don't special-case.
- [ ] WS frame arriving as `Frame.Binary` first: server replies `hello.err("text-required")` and closes.
- [ ] Two concurrent `redeem` of the same code: `ConcurrentHashMap.remove` is atomic, so exactly one wins. Add a dedicated test in `PairingServiceTest`.
- [ ] Clock drift exactly 60 000 ms: boundary test should pin whether this is accepted or rejected (plan says `abs(now - ts) > skewToleranceMs` → accept at 60 000).
- [ ] Second `hello` frame over an already-authenticated connection — server currently ignores unknown `op`, but `ClientHello` op is known. Decide: ignore (silent) vs close. Plan says `onInbound` is called with any typed WireMessage, so a second `ClientHello` goes through `onInbound` → `ApprovalBridge` will reject it as wrong op. OK.

## Complexity / Risk

- **Complexity**: **H** — pure-Kotlin Ed25519 is the bulk of the work (Task 2 is easily half the plan's LOC). Everything else is small, mostly glue.
- **Confidence**: **Medium-High** — spec is crisp (W0 already codified it), Ed25519 has a published reference, Ktor WS test harness is well-documented.
- **Key risks**:
  - **Ed25519 implementation errors** — mitigated by RFC 8032 vectors in Task 1. If vectors don't pass after Task 2 GREEN, block and port from the reference more carefully rather than hack around.
  - **Ktor test-host WS close-race**: occasionally test shuts down before the server sends the final frame. Mitigated by using `incoming.receive()` with a timeout and explicit `close()` from the test side.
  - **iOS `SecRandomCopyBytes`** is compile-only in CI — actual runtime behavior exercised only on device. Plan accepts this for W1.5; a later E2E task validates on a real iPhone.
  - **Coverage on Ed25519 field arithmetic** — some branches (carry propagation on edge scalars) may be hard to reach from RFC vectors alone. Dropped to 70% threshold; a property-based test suite can close the gap in a follow-up if needed.

A one-line follow-up in W1.6 will subscribe `onInbound = { msg -> if (msg is WireMessage.ApprovalResponded) bridge.submitDecision(...) }`; plan leaves this to avoid a cross-plan PR (W1.4 owns the bridge; W1.5 owns the transport).
